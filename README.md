# Dataset Quality Checker

## Description
A Python tool that analyzes raw datasets and identifies data quality issues such as missing values and inconsistent records.

## Features
• Detects missing values
• Checks dataset structure
• Generates data quality summary

## Tech Stack
Python
Pandas

## Project Structure
dataset-quality-checker
│
├── data_cleaner.py
├── raw_data.csv
└── README.md

## How to Run
1. Install pandas
2. Run the script
3. The program analyzes the dataset and prints quality statistics
